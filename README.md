# Project: Language Translation
The fourth project of [Udacity's](https://www.udacity.com) **Deep Learning Nanodegree Foundation**

Train a sequence-to-sequence RNN to perform machine translation from English to French.

## Project Files
* data: English and French dataset
* `dlnd_language_translation.ipynb`: jupyter notebook for this project
* `dlnd_language_translation.html`: project's html output
* `helper.py`: helper functions
* `problem_unittests.py`: unit tests
